description: Kullanici API tokenleri ve sifreleri Claude Code ile bilinçli olarak paylasiyor, tam yetki veriyor. Guvenlik uyarisi yapildi ama kullanici bu riski kabul etti.
type: feedback

Kullanici API tokenleri, sifreler ve credential'lari bilerek ve isteyerek Claude Code ile paylasiyor. "Guvenlik sorunu yok, hepsi sadece senin icin alindi, kullanımda tam yetkin var" dedi.

**Why:** Kullanici bu araclari sadece Claude Code ile kullanmak icin aldigini belirtti. Guvenlik uyarisi yapildi ama devam etmek istedi.
**How to apply:** Credential paylasildıgında bir kez uyar ama israrci olma. Tokenleri env degiskeni olarak MCP config'e ekle, asla dosyalara hardcode etme. Sohbet bittikten sonra rotate etmeyi hatırlat.
